HTML Tags and Their Uses

<h1>
The h1 tag creates the main heading of a webpage.
<p>
The p tag is used to add paragraphs and written information.
<img>
The img tag displays images on a webpage. It uses attributes like src and alt.
<a>
The a tag creates links that connect one webpage to another.
<ul>
The ul tag creates a bulleted list of items.
